Regression Analysis Calculator: How to Use

Use this program to perform a regression analysis of a two-variable data set by drawing either a line or a curve of best fit. Open a text file with the data recorded in the appropriate format (see below). Use the regression menu to select a regression model. The models currently supported are:

  • linear
  • quadratic
  • power (log-log method)
  • exponential (log method).

Quantitative analysis of each variable such as the mean, median, standard deviation, Pearson Correlation Coefficient (r), and coefficient of determination R2 are displayed on the side pane. Click on the eye to see a table of the graphed data, and click 'save' to save an image of the graph with the equation of the regression line or curve as a .jpg or .png. Click 'open' to upload a text file of the data to perform the regression analysis.

How to format the data text file

Please ensure that all uploadad text files for the data are in the following format (Let x represent the independent variable and y represent the dependent variable)

  x Name@y Name@x Unit@y Unit

  x1   y1

  x2   y2

  x3   y3

  ...   ...

Formatted Example:

  MDM4U Mark@Grade12 Mark@%@%

  74   77

  68   66

  92   85

  ...   ...
